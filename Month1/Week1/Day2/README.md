# Day 02: Data Structures

## Overview

This folder contains my solutions and practice exercises completed during **Day 02** of the internship training plan. The focus of this day was understanding Python's built-in data structures and learning how to efficiently store, organize, and manipulate data using lists, tuples, dictionaries, sets, and strings. The day also included solving implementation-based problems to strengthen logical thinking and problem-solving skills.
