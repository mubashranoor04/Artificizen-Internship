# Practice Exercises

## 1. Prime Number Checker

**Objective**

Determine whether a given number is prime.

**Concepts Used**
- Loops
- Conditional Statements
- Modulus Operator
- Boolean Logic

**File:** `primenumber.py`

---

## 2. Reverse a String

**Objective**

Reverse a string without using slicing or Python's built-in reverse functions.

**Concepts Used**
- String Traversal
- String Concatenation
- Loops

**File:** `reverse.py`

---

## 3. Fibonacci Series

**Objective**

Print the Fibonacci sequence up to **n** terms.

**Concepts Used**
- Variables
- Loops
- Sequence Generation

**File:** `Fibonacci.py`

---

## 4. FizzBuzz

**Objective**

Print numbers from **1 to 100** using the following rules:

- Multiples of **3** → `Fizz`
- Multiples of **5** → `Buzz`
- Multiples of **3 and 5** → `FizzBuzz`

**Concepts Used**
- Conditional Statements
- Loops
- Modulus Operator

**File:** `fizzbuzz.py`

---

## 5. Palindrome Checker

**Objective**

Check whether a given string is a palindrome.

**Concepts Used**
- String Manipulation
- Comparison Operations
- Loops

**File:** `palindrome.py`

---

## 6. Largest and Smallest Number

**Objective**

Find the largest and smallest values in a list without using `min()` or `max()`.

**Concepts Used**
- Lists
- Traversal
- Comparison Logic

**File:** `min_max.py`
